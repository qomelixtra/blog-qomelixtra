# សួស្តី ពិភពលោក!

សូមស្វាគមន៍មកកាន់ប្លុកនេះ! នេះជាអត្ថបទទីមួយរបស់ខ្ញុំជាភាសាខ្មែរ។

## ប្លុកនេះនិយាយអំពីអ្វី?

ប្លុកនេះនឹងនិយាយអំពី៖

- ការអភិវឌ្ឍន៍គេហទំព័រ
- JavaScript និង TypeScript
- React និង Next.js
- គន្លឹះល្អៗនានា

## កូដគំរូ

នេះគឺជា function សាមញ្ញមួយក្នុង JavaScript៖

```javascript
function សួស្តី(ឈ្មោះ) {
  return `សួស្តី ${ឈ្មោះ}!`;
}

console.log(សួស្តី("ពិភពលោក"));
```
